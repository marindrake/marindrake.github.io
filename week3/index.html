    Object Methods: this

    1. Functions that are stored in object properties are called “methods”.
    2. Methods allow objects to “act” like object.doSomething().
    3. Methods can reference the object as this.

      4. The value of this is defined at run-time.

    4. When a function is declared, it may use this, but that this has no value until the function is called.
    5. A function can be copied between objects.
    6. When a function is called in the “method” syntax: object.method(), the value of this during the call is object.

    Ch5: Objects

    1. Objects are a collection of key-value pairs placed inside curly braces {}.
    2. Objects have properties that can be any JavaScript value. If it’s a function, it’s known as amethod.
    3. An object’s properties and methods can be accessed using either dot notation or square bracket notation.
    4. Objects are mutable, which means their properties and methods can be changed or removed.
    5. Objects can be used as parameters to functions, which allows arguments to be entered in any order, or omitted.
    6. Nested objects can be created by placing objects inside objects.
    7. JSON is a portable data format that uses JavaScript object literals to exchange information.
    8. The Math object gives access to a number of mathematical constants.
    9. The Math object can be used to perform mathematical calculations.
    10. The Date object can be used to create date objects.
    11. Once you’ve created a Date object, you can use the getter methods to access information about that date.
    12. Once you’ve created a Date object, setter methods can be used to change information about that date.
    13. The Regex object can be used to create regular expressions.

    Ch6: Document Object Model

    1. The Document Object Model is a way of representing a page of HTML as a tree of nodes.
    2. The document.getElementById() , document.getElementsByClassName() , document.getElementsByTagNames() and document.querySelector() can be used to access elements on a page.
    3. The parentNode() , previousSibling() , nextSibling() , childNodes() and children() methods can be used to navigate around the DOM tree.
    4. An element’s attributes can be accessed using the getAttribute() method, and updated using the setAttribute() method.
    5. The createElement() and createTextNode() methods can be used to create dynamic markup on the fly.
    6. Markup can be added to the page using the appendChild() and insertBefore() methods.
    7. Elements can be replaced using the replaceChild() method, and removed using the removeChild() method.
    8. The innerHTML property can be used to insert raw HTML directly into the DOM.
    9. The CSS properties of an element can be changed by accessing the style property.
    10. Function declarations can be invoked before they are defined because they are hoisted to the top of the scope, but function expressions cannot be invoked until after they are defined.
