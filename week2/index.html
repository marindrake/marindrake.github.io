    Ch3: Arrays, Logic, and Loops

    1. Arrays are an ordered list of values.
    2. Multidimensional arrays are arrays that contain other arrays.
    3. Arrays have lots of methods that can be used to manipulate items in the array.
    4. Sets are new in ES6 and are ordered lists of non-duplicate value.
    5. Maps are new in ES6 and are ordered lists of key-value pairs.
    6. We can use an if and else statement to control the flow of code.
    7. The switch statement can be used instead of multiple if and else statements.
    8. A while loop and do ... while loop can be used to repeat a block of code while a condition is still true.
    9. A for loop works in a similar way to a while loop, but has a different syntax.
    10. A for-of loop can be used to iterate over an array.
    11. Sets and maps are enumerable, so can also be looped over using a for-of loop.

    Ch4: Functions

    1. Functions are first-class objects that behave the same way as other values.
    2. Function literals can be defined using the function declaration, or by creating afunction expressionby assigning an anonymous function to a variable.
    3. All functions return a value. If this is not explicitly stated, the function will return undefined.
    4. A parameter is a value that is written in the parentheses of a function declaration and can be used like a variable inside the functionâ€™s body.
    5. An argument is a value that is provided to a function when it is invoked.
    6. The arguments variable is an array-like object that allows access to each argument provided to the function using index notation.
    7. The rest operator can be used to access multiple arguments as an array.
    8. Default arguments can be supplied to a function by assigning them to the parameters.
    9. Arrow functions are a new shorthand notation that can used for writing anonymous functions in ES6.
    10. Function declarations can be invoked before they are defined because they are hoisted to the top of the scope, but function expressions cannot be invoked until after they are defined.
    11. A callback is a function that is provided as an argument to another function.
